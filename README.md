# Mélanie Dupriez — Business Data Analyst & Développeuse

Je transforme les données complexes en leviers stratégiques grâce au **Data Storytelling** et à des **interfaces web sur mesure**.

## 🚀 À propos
Je suis Business Data Analyst et je fais le pont entre la donnée brute et la prise de décision. Je ne me contente pas d'analyser les données ; je conçois les outils pour les visualiser et les récits pour les expliquer. Mon objectif est d'aider les entreprises à prendre des décisions éclairées grâce à un reporting à fort impact et des interfaces interactives personnalisées.

## 💡 Ma valeur ajoutée
* **Dashboard Storytelling** : Conception de rapports intuitifs qui transforment des métriques complexes en insights clairs, prêts pour la prise de décision.
* **Interfaces Web de données sur mesure** : Développement d'applications web personnalisées pour rendre la visualisation de données accessible, interactive et visuellement impactante, dépassant les standards des outils BI classiques.
* **Analyse orientée Business** : Chaque projet — qu'il s'agisse d'une interface ou d'un rapport — est pensé pour servir un objectif stratégique précis pour l'entreprise.

## 🛠️ Stack Technique
* **Data & BI** : SQL, Python, Power BI, Tableau.
* **Développement Web** : HTML5, Tailwind CSS, JavaScript (Animations personnalisées, composants interactifs).
